# Prerequisites

## Emscripten

1. Install Emscripten SDK:
```
git clone https://github.com/emscripten-core/emsdk.git
cd emsdk
```

2. Download and install the latest SDK tools:
```
./emsdk install latest
./emsdk activate latest
```

## http-server
For testing
```
npm i -g http-server
```
